# Netacademia Programozás JavaScriptben tanfolyam

https://netacademia.hu/Tanfolyam/programozas-javascriptben

Az elsõ rész témája a JavaScript nyelv bemutatása az alapoktól kezdve, a rejtélyes trükkökön át az egészen professzionális tervezési mintákig. A képzés legfontosabb célja annak bemutatása, hogy a JavaScript olyan, mint Pandora az Avatarban. Nem egyszerû földi nyelv ez, mint a C++, vagy a Java, mivel teljesen más gondolkodásmódot igényel a használata. Az utóbbi miatt a nyelvi sajátosságok mellett azok miértjeire is kitérünk.
